# JSON

You should use the following to use the schema:

```json
{
    $schema: "https://twopizza9621536.github.io/schema/json/${schema}",
    "datatype1": [],
    "datatype2": -1,
    "datatype3": "..."
}
```

Available schemas are in the repository.
